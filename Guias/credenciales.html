Credenciales

Obtener credenciales en un ordenador es un ataque en ciberseguridad donde se busca acceder a las credenciales de usuario, como nombres de usuario y contraseñas. Esto se puede lograr mediante técnicas como el keylogging (registrar las pulsaciones del teclado), phishing (suplantación de identidad), exploiting (aprovechamiento de vulnerabilidades) o utilizando herramientas como Mimikatz para extraer credenciales almacenadas en la memoria del sistema. Es crucial implementar medidas de seguridad como autenticación multifactor, actualizaciones regulares y concienciación sobre amenazas para prevenir este tipo de ataques.

Ejemplos:

Linux:

El primer paso es obtener una sesión de Meterpreter.

Con la sesión Meterpreter obtenida, podemos utilizar el módulo linux/gather/hashdump para obtener un volcado de hash añadiendo como parámetro la sesión obtenida.

Después, podemos tratar de crackearlos fuera o con otro módulo auxiliar de crackeo.

Windows:

El primer paso es obtener una sesión de Meterpreter.

Con la sesión Meterpreter obtenida, podemos utilizar Kiwi para intentar conseguir las credenciales en un sistema Windows.

Para ello, una vez tengamos una Shell Meterpreter, lanzamos load kiwi para cargar el módulo y, una vez en marcha, utilizamos creds_all para obtener los hash de las sesiones abiertas.

Estos los podemos intentar crackear fuera o utilizando algún módulo de descifrado dentro de Metasploit.
